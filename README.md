# Walmart-Sales-Analysis

This project analyzes Walmart store sales using advanced MySQL techniques like complex queries, CTEs, window functions, and aggregation. It also includes an interactive **Power BI dashboard** for visual sales insights.

---

## 📊 Power BI Sales Dashboard

An interactive Power BI report was built to visualize key sales metrics across branches, cities, product lines, and customer segments for **January – March 2019**.

### Dashboard Preview

![Walmart Sales Dashboard](Sales%20Dashboard.PNG)

### Key Metrics
| Metric | Value |
|---|---|
| Total Sales | 322.97K |
| Total Profit | 15.38K |
| Total Customers | 1K |
| Average Rating | 6.97 |

### Dashboard Highlights
- **Monthly Sales Trend** – January highest (116K), March (109K), February (97K)
- **Sales by Product Line** – Food & Beverages led at 56K
- **Sales by City** – Naypyitaw (111K) topped Yangon & Mandalay (106K each)
- **Sales by Branch** – Branch C (111K) was top; A and B tied at 106K
- **Customer Type** – Members (50.85%) vs Normal (49.15%)
- **Payment Methods** – E-wallet most preferred (345 transactions)
- **Gender-wise Sales** – Female 167.88K vs Male 155.08K
- **Product Rating** – Food & Beverages rated highest (7.1)

### Filters Available
- Month | City | Branch | Gender
